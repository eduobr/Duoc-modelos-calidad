Task: Identificar y Analizar Necesidades de Entrenamiento
Relationships
RolesPrimary Performer:

Additional Performers:
Outputs
  • Plan de entrenamiento organizacional

    Main Description
    Se identifican y analizan las necesidades de entrenamiento que es necesario suplir para aumentar la eficiencia y efectividad del personal. Estas necesidades provienen tanto a nivel organizacional (por RRHH) así como también a nivel de proyecto (por el Jefe de proyecto). Refiérase a tarea Generar Plan de Proyecto del área de procesos Planificación del Proyecto para planificar entrenamiento para el proyecto.
