Task: Ejecutar Entrenamiento
Relationships
RolesPrimary Performer:
  • Grupo de ingeniería de procesos
  • Jefe de proyecto
Additional Performers:
InputsMandatory:
  • Lista de participantes de entrenamiento
  • Plan de proyecto
    		Optional:
    • None
    Outputs
    • Catálogo de recursos de entrenamiento
    • Registro de resultados de entrenamiento
      Main Description
      Seleccionado el personal a ser entrenado, se ejecuta el proceso de entrenamiento según se estableció en la planificación. Existen varios métodos mediante los cuales se puede llevar a cabo esta tarea, como por ejemplo: manuales, curso de capacitación interno (guiado por algún funcionario experto), curso de capacitación externo (refiérase a área de procesos Administración de Acuerdos con Proveedores), u otro.
      Para todos los tipos de entrenamiento se lleva un registro de asistencia de los participantes, con tal de tomar acciones correctivas en el caso que un empleado no cumpla con requisitos mínimos de asistencia.
      En caso de generarse material que pueda ser utilizado para futuras capacitaciones, este es registrado en el “Catálogo de recursos de entrenamiento”.
