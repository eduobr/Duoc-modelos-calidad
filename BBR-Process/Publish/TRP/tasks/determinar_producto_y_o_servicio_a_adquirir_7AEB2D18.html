Task: Determinar Producto y/o Servicio a Adquirir
Relationships
RolesPrimary Performer:
  • Jefe de proyecto
Additional Performers:
InputsMandatory:
  • Especificación de Requerimientos de Software (ERS)
    		Optional:
    • None
    Outputs
    • Plan de proyecto
    • Solicitud de cotización
      Main Description
      Se debe determinar el tipo de productos y/o servicios a adquirir. Esto es, determinar qué partes del proyecto serán subcontratadas, así como los productos de hardware y software a adquirir.

      Se realiza una revisión de los requerimientos para determinar cuáles de estos pueden ser realizados por terceros.

      Las partes del proyecto que serán subcontratadas deben documentarse en el “Plan de administración de proveedores y subcontratistas” del artefacto “Plan de proyecto”.

      A menos de tratarse de Proveedores predefinidos, deberán realizarse al menos tres cotizaciones previas a la adquisición de cualquier producto y/o servicio.
