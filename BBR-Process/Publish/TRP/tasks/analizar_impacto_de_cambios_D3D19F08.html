Task: Analizar Impacto de Cambios
Relationships
RolesPrimary Performer:
  • Analista
  • Cliente
  • Jefe de proyecto
Additional Performers:
InputsMandatory:
  • Solicitud de requerimientos
    		Optional:
    • None
    Main Description
    Todo cambio solicitado debe cumplir con los criterios de aceptación definidos para el proyecto. Estos cambios deben tener una revisión de factibilidad e impacto. El Analista estudia el impacto del cambio del requerimiento o del nuevo requerimiento. El mismo es comentado al Jefe de proyecto, el cual resuelve con el Cliente para ver la factibilidad de la implantación del cambio.
