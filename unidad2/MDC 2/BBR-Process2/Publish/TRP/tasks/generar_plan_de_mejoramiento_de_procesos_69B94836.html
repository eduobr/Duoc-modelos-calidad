Task: Generar Plan de Mejoramiento de Procesos
Relationships
RolesPrimary Performer:
  • Grupo de ingeniería de procesos
Additional Performers:
InputsMandatory:
  • Plan de mejoramiento de procesos
    		Optional:
    • None
    Outputs
    • Plan de mejoramiento de procesos
      Main Description
      Se deben determinar las necesidades organizacionales para mejoramiento de procesos evaluado cómo se han comportando los procesos organizacionales y generar el artefacto “Plan de mejoramiento de procesos” anual, donde se planifican las mejoras que se realizarán durante el período.
      Se debe considerar además como entrada las mejoras que no hayan podido concretarse en el período anterior validando que éstas sigan siendo un aporte para la Organización.
