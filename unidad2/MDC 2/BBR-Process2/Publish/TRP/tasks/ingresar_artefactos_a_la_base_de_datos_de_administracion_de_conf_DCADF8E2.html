Main Description
Ingresar los artefactos catalogados a la BDAC generando la primera línea base del proyecto.

El repositorio del proyecto puede elaborarse de varias formas. Entre las más frecuentes se encuentran: 1) una instancia separada de repositorio durante el desarrollo, desde el cual se suben líneas bases a otro repositorio formal que contiene versiones liberadas por el proyecto. 2) un desarrollo en un medio de trabajo que hace promociones en la misma biblioteca según hitos (por ej. en SVN es trunk, branches, tags).
