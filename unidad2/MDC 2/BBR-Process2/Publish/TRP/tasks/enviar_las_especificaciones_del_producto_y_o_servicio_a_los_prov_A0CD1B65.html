Task: Enviar las Especificaciones del Producto y/o Servicio a los Proveedores
Relationships
RolesPrimary Performer:
  • Gerente de proyectos
Additional Performers:
InputsMandatory:
  • Especificación de Requerimientos de Software (ERS)
  • Lista de proveedores
    		Optional:
    • None
    Main Description
    Se hace efectivo el envío a los posibles Proveedores de la propuesta del producto y/o servicio a adquirir, a través del documento “Especificación de Requerimientos de Software (ERS)” asociado al producto y/o servicio. Este indica los requerimientos del producto y/o servicio a adquirir y las condiciones bajo las cuales se evaluará el producto y/o servicio.
